# North Star Investigation Bootstrap

## Goal

Create or refresh the Commander's Intent doc so the next operator can run the first brutal test without reconstructing the problem.

## Required work

1. Normalize the investigation doc.
2. Write the North Star, scope, and non-negotiables.
3. Define the scoreboard.
4. Capture the ground-truth anchors.
5. Write the quant model or sanity checks.
6. Rank the hypotheses.
7. Propose the first 1-3 bets.
8. Seed the worklog with what you inspected and what you currently believe.

## Bootstrap examples

- Good:
  - "If p95 is dominated by service X, the test is to bypass X and compare the delta."
  - "If the attrition model assumes 20 percent drop here, write the sanity check that proves or falsifies that."
- Weak:
  - "Look around and see what seems off."
