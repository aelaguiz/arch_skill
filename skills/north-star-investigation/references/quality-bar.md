# North Star Investigation Quality Bar

## Quant model

- Strong:
  - names what is measured vs assumed
  - uses simple arithmetic or sanity checks to bound the space
- Weak:
  - purely narrative theory
  - fake precision with unsupported numbers

## Ranked hypotheses

- Strong:
  - ordered by current evidence
  - easy to invalidate or strengthen with the next test
- Weak:
  - unordered list of possibilities
  - stale theories kept alive after disconfirming evidence

## Brutal tests

- Strong:
  - can falsify a theory quickly
  - have a pre-committed decision rule
  - minimize implementation overhead
- Weak:
  - broad observability work first
  - "we'll know it when we see it"
