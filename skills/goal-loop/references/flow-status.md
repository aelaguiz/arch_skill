# Goal Loop Flow-Status

## Goal

Inspect the controller and worklog read-only and recommend the single best next move.

## What to inspect

- North Star confirmation
- scoreboard completeness
- lever inventory
- latest best belief
- latest worklog entry
- whether the next bet is already obvious

## Recommendation outcomes

- repair the controller first
- confirm the North Star
- run the next iteration
- write a context digest
- retire or complete the loop

## Output rule

- Keep it short.
- Name the current best belief.
- Name the biggest remaining uncertainty.
- Name the exact next move.
