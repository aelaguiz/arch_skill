# Goal Loop Setup Mode

## Goal

Create or repair a controller that can survive restarts and guide future iterations.

## Required work

1. Normalize the controller structure.
2. Draft the North Star and scoreboard.
3. List the main levers.
4. Write the anti-sidetrack contract.
5. Define the iteration protocol and de-dupe rule.
6. Seed `WORKLOG_PATH` if it does not exist.

## Stop condition

- New or still-draft controllers stop for North Star confirmation before active iteration begins.
- If the goal itself is ambiguous, do not fake momentum with a first bet.
