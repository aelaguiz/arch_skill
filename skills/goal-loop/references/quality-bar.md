# Goal Loop Quality Bar

## Controller

- Strong:
  - North Star is concrete
  - scoreboard is measurable
  - lever inventory is actionable
  - anti-sidetrack contract is explicit
- Weak:
  - goal is aspirational but not measurable
  - no clear levers
  - controller reads like a brainstorming note

## Bets

- Strong:
  - one clear lever
  - one decision rule
  - realistic information gain
- Weak:
  - disguised implementation project
  - several experiments bundled together
  - no pre-committed threshold

## Worklog entries

- Strong:
  - exactly one iteration
  - evidence and result are concrete
  - conclusion updates the current best belief
- Weak:
  - multiple partial bets in one entry
  - summary with no evidence
  - next step detached from what was learned
